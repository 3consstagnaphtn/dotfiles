## How to install ncmpcpp on Android
```bash
# Installs the ncurses MPD client
$ pkg install ncmpcpp
# Install Music Player Daemon
$ pkg install mpd
# installs the command line client for MPD
$ pkg install mpc
```
Then you are good to go. Just run ```ncmpcpp``` and then enjoy!
